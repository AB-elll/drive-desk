# DriveDesk

Google Drive監視型ドキュメント自動処理パイプライン

> Watch your Drive. Process everything. Automatically.

## 概要

DriveDesKは、指定したGoogle Driveフォルダを監視し、流入するあらゆるファイルをAIが理解・分類し、会計ソフトや各種管理ツールへ自動反映するOSSパイプラインです。

```
[LINE / メール / 手動 / 他ツール]
           ↓
    【Google Drive】← 万能インボックス
           ↓
      【DriveDesk】
           ↓
  [freee / スプレッドシート / Slack / 他]
```
